# Instructions to run

## Cloning the repo
```
$ git clone https://github.com/CodeChefVIT/Simple-CRUD
$ cd Simple-CRUD
$ cd GraphQL
```
## Downloading NPM packages
```
$ npm i

```

## Create a .env file
```
MONGODB_URL = <database URL>
```
## Running the project
```
$ npm run dev

```

## Viewing Playground
```
- Go to localhost:3000/playground

```
